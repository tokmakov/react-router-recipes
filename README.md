# Рецепты блюд всего мира

Небольшое приложение — каталог рецептов блюд всего мира, не имеет практической ценности, сделано исключительно с целью изучения React. Для оформления используется css-фреймворк materialize.css, http-запросы на получение рецептов отправляются сервису themealdb.com. Подробности [здесь](https://tokmakov.msk.ru/blog/item/652), демо-сайт [здесь](http://react-router-recipes.tokmakov.msk.ru/).
